# 🏎️ Detailing Center Management System (v1.0.0)

Информационная система для автоматизации процесса онлайн-записи клиентов детейлинг-центра. Проект разработан как финальный этап (ЛР №6) и подготовлен к защите в рамках выпускной квалификационной работы.

---

### 📝 1. Описание проекта
Приложение позволяет пользователям просматривать перечень доступных услуг детейлинг-центра, выбирать удобное время для записи и отслеживать статус своих заказов в личном кабинете. Система автоматизирует взаимодействие между клиентом и администратором, исключая необходимость ручной обработки первичных заявок.

### 🎯 2. Цели и задачи разработки
* **Основная цель:** Завершение разработки программного продукта и подготовка его к представлению в рамках проектной деятельности.

### 💻 3. Стек технологий
Проект реализован на современном технологическом стеке, обеспечивающем высокую производительность и масштабируемость:
* **Frontend:** React (Vite)
* **Backend:** PHP 8.x (REST API).
* **База данных:** MySQL (использование PDO для безопасности).
* **Архитектура:** Feature-Sliced Design (FSD).
* **Инструменты:** Docker (контейнеризация), Composer, PlantUML (моделирование).

На бэкенде используется сервисно-ориентированный подход с единой точкой входа (`index.php`), обрабатывающей запросы через контроллер.

### ⚙️ 5. Инструкция по развертыванию

#### Подготовка базы данных
1.  Создайте базу данных в MySQL.
2.  Импортируйте предоставленный SQL-дамп (структура таблиц `users`, `services`, `bookings`).
3.  Отредактируйте параметры подключения в файле `api/db.php`.

#### Запуск Backend
1.  Перенесите папку `api` в рабочую директорию вашего веб-сервера (XAMPP, OpenServer или Docker-контейнер).
2.  Убедитесь, что сервер поддерживает PHP 8.0+.

#### Запуск Frontend
1.  Перейдите в папку клиента:
    ```bash
    cd client
    ```
2.  Установите необходимые зависимости:
    ```bash
    npm install
    ```
3.  Запустите сервер разработки:
    ```bash
    npm run dev
    ```
